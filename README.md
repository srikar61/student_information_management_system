# SIMS (Student Information Management System) MEAN App

## Overview

SIMS is a comprehensive Student Information Management System designed using the MEAN stack (MongoDB, Express, Angular, and Node.js). This web application allows various roles—Admin, Parent, and Student—to manage their respective information and access specific features based on their role. It provides a dashboard for each role, facilitating easy access to notifications, assignments, bus details, and more.
